1. OVERVIEW

Kanakku template is built with Laravel 9.11 with Laravel Mix provides a clean build steps for your Laravel application. Mix supports several common CSS and JavaScript pre-processors. We have also added few reusable components which speedup your development.

1. 1. Features

  • Creative Design: Kanakku is a bold theme that organizing content to follow natural eyes, striking balance between beautiful and visual hierarchy.
  • Fully Responsive: The theme passed Mobile-Friendly Test – by Google. We also tested carefully the theme on almost devices: laptop, tablet, and phone. All elements are 100% responsive and ready for all devices.
  • Pixel Perfect Design
  • Color & Fonts is easily changed
  • Customer Reviews, Customer accounts, Compare Products, Grid & List layout, product fillter
  • Newsletter, Social Sharing
  • Bootstrap 5.x used
  • Compatible Browsers: IE8+, Firefox, Safari, Opera, Edge, Chrome

1. 2. Installation Guide

  • Open the terminal in your root directory(Kanakku _laravel) & to install the composer packages run the following command:

composer install

  • In the root directory, you will find a file named .env.example, copy the content,create a new file and paste that content save as .env and run the following command to generate the key.

php artisan key:generate

  • Open the terminal in your root directory(Kanakku _laravel) & and run the following command for installing plugins used in the application:

npm install

  • Open the terminal in your root directory(Kanakku _laravel) & and run the application as a below command:

npm run dev

  • Now you can run this Template in any web browser from your PC or Laptop.

LOGIN ROUTE:

For example: localhost/Kanakku_laravel/blue/public/login or www.example.com/Kanakku_laravel/blue/public/login

REGISTER ROUTE:

For example: localhost/Kanakku_laravel/blue/public/registration or www.example.com/Kanakku_laravel/blue/public/registration
