1. OVERVIEW

Kanakku is a well designed fully responsive admin template that makes the web administration process easier than ever. It is equipped with various functionalities and is platform compatible.

1. 1. Features

  • Creative Design: Kanakku is a bold theme that organizing content to follow natural eyes, striking balance between beautiful and visual hierarchy.
  • Fully Responsive: The theme passed Mobile-Friendly Test – by Google. We also tested carefully the theme on almost devices: laptop, tablet, and phone. All elements are 100% responsive and ready for all devices.
  • Pixel Perfect Design
  • Color & Fonts is easily changed
  • Customer Reviews, Customer accounts, Compare Products, Grid & List layout, product fillter
  • Newsletter, Social Sharing
  • W3C Validated Code
  • Bootstrap 5.x used
  • Compatible Browsers: IE8+, Firefox, Safari, Opera, Chrome

1. 2. Installation Guide

    Node.js and NPM:

  • If NodeJS is not installed then download it from below link and NPM comes bundled with Node.js

    • Note : Please make it sure your node version 16.17.0

https://nodejs.org/

    Angular-cli:

  • You can install angular-cli by executing from your command below comand

    • Note : Please make it sure your Angular-cli version 15.0.0

npm install -g @angular/cli

    Update it to the latest Angular CLI:

  • IF you already installed,uninstalled it and again install the angular cli.

npm uninstall --global @angular/cli

npm install --global @angular/cli@15.0.0

    NPM Packages:

  • Before proceding you'll need to install npm packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

npm install

    Development Server:

  • Run ng serve for a dev server. Navigate to localhost:4200/. The app will autometically reload if you change any of the source files.

ng serve

    Build Project:

  • Run the project as per below command. The build artifacts will be stored in the dist/ directory. Use the --prod flag for a production build.

ng build

    More Help:

  • To get more help on the angular-cli use below command or go check out the Angular-CLI README.

ng --help
